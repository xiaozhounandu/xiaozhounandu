# 小猪南都数据看板 - 真实数据管理说明

## 📊 概述

已为小猪南都客户管理系统添加了完整的数据管理功能，支持真实数据的添加和管理，让数据看板展示真实的业务数据。

## 🎯 主要功能

### 1. 数据管理页面
- **访问路径**: `/data-management`
- **权限要求**: 管理员(ADMIN)或销售经理(MANAGER)
- **功能**: 批量生成测试数据，管理用户和客户信息

### 2. 数据添加功能

#### 👤 添加测试用户
- **功能**: 快速创建系统用户
- **支持角色**: 管理员、销售经理、销售员
- **自动生成**: 邮箱和手机号
- **用途**: 创建销售团队成员账号

#### 🏢 批量添加客户
- **功能**: 一次性生成多个测试客户
- **数量范围**: 1-100个
- **随机属性**: 行业、等级、状态、归属人
- **用途**: 快速丰富客户数据库

#### 📝 添加跟进记录
- **功能**: 为特定客户创建跟进记录
- **跟进方式**: 电话、拜访、邮件、微信、会议
- **自动设置**: 7天后的下次跟进时间
- **用途**: 模拟真实的销售跟进过程

### 3. 数据统计展示
- **实时更新**: 数据添加后立即反映在看板中
- **数据概览**: 用户数、客户数、跟进记录数
- **操作记录**: 显示所有操作的成功/失败状态

## 📊 数据结构说明

### 用户数据
```sql
-- 用户角色
ADMIN: 管理员 - 拥有所有权限
MANAGER: 销售经理 - 可查看统计数据，管理团队成员
USER: 销售员 - 只能管理自己名下的客户
```

### 客户数据
```sql
-- 客户等级
A: VIP客户 - 高价值客户
B: 重要客户 - 中等价值客户
C: 普通客户 - 一般价值客户
D: 低价值客户 - 待开发客户

-- 客户状态
POTENTIAL: 潜在客户 - 初期接触
CONTACTED: 已联系 - 建立联系
NEGOTIATING: 谈判中 - 商务谈判
ACTIVE: 活跃客户 - 持续合作
SUCCESS: 已成交 - 完成交易
FAILED: 已流失 - 合作中断
```

### 跟进数据
```sql
-- 跟进方式
phone: 电话沟通
visit: 实地拜访
email: 邮件联系
wechat: 微信沟通
meeting: 会议讨论
```

## 📈 数据看板指标

### 核心指标
- **总客户数**: 系统中所有客户总数
- **新增客户**: 最近30天新增的客户数
- **活跃客户**: 当前状态为ACTIVE的客户
- **成交客户**: 已完成交易的客户
- **流失客户**: 合作中断的客户

### 统计图表
- **近7天趋势**: 每日新增客户数量
- **行业分布**: 不同行业客户占比
- **等级分布**: A/B/C/D级客户分布
- **月度趋势**: 全年客户增长情况

### 跟进统计
- **今日跟进**: 今天计划的跟进任务
- **待跟进**: 即将进行的跟进记录

## 📱 使用建议

### 数据生成策略
1. **先创建用户**: 建立完整的销售团队
2. **分批生成客户**: 每次生成20-50个客户
3. **模拟跟进过程**: 为不同客户添加跟进记录
4. **定期维护**: 更新客户状态和跟进信息

### 数据质量
- 确保客户信息完整性
- 维护真实的跟进记录
- 定期更新客户状态
- 保持数据的一致性

### 权限管理
- 管理员拥有所有权限
- 销售经理可查看团队数据
- 销售员只能管理自己的客户
- 敏感操作需要相应权限
